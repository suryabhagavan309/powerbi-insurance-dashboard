# Insurance Claims & Policy Analysis Dashboard

## 📊 Overview
This project analyzes insurance policies, claims, and customer data using Power BI.

## 🔍 Key Insights
- Majority of policies are active
- Travel and health policies generate higher premiums
- Most claims are settled successfully
- Adults contribute higher claim amounts

## 🛠 Tools Used
- Power BI
- Data Modeling
- Data Visualization
